
# PEBBLE CLONE 
Welcome to the GitHub repository for the Pebble Clone project!
#
**Overview**:
Pebble Clone is a project developed using React.JS for the front-end, incorporating react-router-dom for seamless routing functionality. To ensure a visually appealing interface, we utilized Bootstrap alongside custom SCSS. The site is fully responsive, ensuring a seamless user experience across various devices.
#
**How to Use:**

1. Clone the repository to your local machine.
2. Node.js should be installed on local machine.
3. Run command **__npm start__** in Terminal
